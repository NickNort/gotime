# Black-box tests for cmd/climain.go

## Scope and goals

- Validate CLI behaviors without refactoring:
  - Usage/help when no positional content; non-zero exit
  - Successful run creates `qr.svg` with required colors (#552048 fg, #f8f2ec bg)
  - Flag handling for long/short, with `=` and space forms, and flags after positional arg
  - Error paths: invalid values and missing flag values

## Approach

- Build the CLI once in TestMain into a temp binary, then execute it in tests (exec per scenario).
- Each test runs the binary in its own temp working directory so `qr.svg` is isolated and easily validated.
- Capture stdout/stderr and exit codes; avoid `t.Parallel` to keep runs deterministic.
- Windows support: append `.exe` to the built path when `runtime.GOOS == "windows"`.

## Implementation steps

1. Create `cmd/climain_test.go` with a `TestMain(m *testing.M)` that:

   - Resolves repo root (dir of this test, `..`), builds the CLI: `go build -o <tmp>/climain[.exe] ./cmd/climain.go`.
   - Stores a package-level `cliPath` used by all tests.

2. Add helper `runCLI(t, workDir string, args ...string)` to execute the built binary:

   - Sets `cmd.Dir = workDir`, captures stdout/stderr, returns exit code and outputs.

3. Test: no args prints usage and exits with code 1; no `qr.svg` created.
4. Test: valid run `gotime-cli "hello"` creates `qr.svg` containing `<svg`, `#552048`, and `#f8f2ec`.
5. Test: flags after positional arg (e.g., `"hello" -c circle -f rounded -m diamond`) succeed and create `qr.svg`.
6. Test: long/short flag variants with `=` and space forms (e.g., `-finder-center=circle`, `-c circle`, etc.).
7. Test: invalid flag value (e.g., `-finder-center triangle`) yields non-zero exit and stderr mentioning allowed values.
8. Test: missing flag value (e.g., `-c` at end or before another flag) yields error to stderr and non-zero exit.

## Sample harness skeleton (for clarity)

```go
package main_test

import (
    "bytes"
    "os/exec"
    "path/filepath"
    "runtime"
    "testing"
)

var cliPath string

func TestMain(m *testing.M) {
    // build CLI into temp dir and set cliPath...
}

func runCLI(t *testing.T, workDir string, args ...string) (int, string, string) {
    t.Helper()
    cmd := exec.Command(cliPath, args...)
    cmd.Dir = workDir
    var out, errb bytes.Buffer
    cmd.Stdout = &out
    cmd.Stderr = &errb
    runErr := cmd.Run()
    code := 0
    if runErr != nil {
        if ee, ok := runErr.(*exec.ExitError); ok {
            code = ee.ExitCode()
        } else {
            t.Fatalf("failed to run: %v", runErr)
        }
    }
    return code, out.String(), errb.String()
}
```
